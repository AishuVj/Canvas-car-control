# Canvas Car Control Game

A browser-based car control game built using HTML5 Canvas and JavaScript.

## Features
- User-controlled car direction and speed
- Dynamic color customization
- Responsive button controls

## Technologies
- HTML
- CSS
- JavaScript (Canvas API)

## Live Demo
[Play the game here](https://aishuvj.github.io/Canvas-car-control)

## How to Run Locally
1. Clone the repository
2. Open `index.html` in your browser
